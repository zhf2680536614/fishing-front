# 高德地图地理编码测试

本文件夹包含测试高德地图地理编码功能的脚本。

## 测试脚本

- `test-geocode.html`：测试高德地图地理编码功能的 HTML 页面

## 如何使用

1. **获取高德地图安全密钥**：
   - 登录高德地图开放平台：https://console.amap.com/
   - 进入"应用管理" -> "我的应用"
   - 找到您的应用，点击"查看"或"设置"
   - 在"安全密钥"部分，点击"获取安全密钥"或"生成安全密钥"
   - 复制生成的安全密钥（jscode）

2. **配置安全密钥**：
   - 打开 `test-geocode.html` 文件
   - 找到 `const AMapSecret = '';` 这一行
   - 将获取到的安全密钥粘贴到引号中，例如：`const AMapSecret = '您的安全密钥';`

3. **运行测试**：
   - 打开 `test-geocode.html` 文件
   - 点击 "开始测试" 按钮
   - 查看测试结果

## 测试内容

1. **正向地理编码**：将地址（如 "重庆市渝北区"）转换为经纬度
2. **逆向地理编码**：将经纬度转换为地址

## 注意事项

- 确保您的 API Key 是 Web 端类型，并且有足够的调用配额
- **必须同时配置 API Key 和安全密钥（jscode）才能使用地理编码功能**
- 确保您的网络连接正常，能够访问高德地图 API 服务器
- 如果测试失败，请检查控制台中的错误信息

## 安全密钥的作用

高德地图为了增强 API 安全性，要求同时使用 API Key 和安全密钥（jscode）：
- 防止 API Key 被滥用
- 提高调用的安全性
- 确保只有授权的应用才能使用 API
